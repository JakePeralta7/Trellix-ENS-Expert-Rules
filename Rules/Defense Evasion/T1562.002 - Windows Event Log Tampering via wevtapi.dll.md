# T1562.002 - Windows Event Log Tampering via wevtapi.dll

## Author
Elad Levi

## Description

## Rule Class
Processes

## Rule TCL
```tcl
Rule {
	Initiator {
		Match PROCESS {
			Include OBJECT_NAME {
				-v "**"
			}
			Exclude VTP_PRIVILEGES -type BITMASK { -v 0x8 }
			Exclude PROCESS_CMD_LINE {
				-v "C:\\Windows\\system32\\svchost.exe -k wsappx"
			}
		}
    }
	Target {
		Match SECTION {
			Include OBJECT_NAME { 					
				-v "wevtapi.dll"
			}
			Include -access "EXECUTE"
		}
	}
}
```
