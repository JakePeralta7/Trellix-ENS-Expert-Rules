# T1562.002 - Windows Event Log Tampering via wevtapi.dll

## Author
Elad Levi

## Description

## Rule Class
Processes

## Rule TCL
```tcl
Rule {
	Initiator {
		Match PROCESS {
			Include OBJECT_NAME {
				-v "**"
			}
			Exclude VTP_TRUST true
		}
    }
	Target {
		Match SECTION {
			Include OBJECT_NAME { 					
				-v "wevtapi.dll"
			}
			Include -access "EXECUTE"
		}
	}
}
```
