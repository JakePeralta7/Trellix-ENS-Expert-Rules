# T1518.001 - Aacinfo.exe Usage

## Author
Elad Levi

## Description

## Rule Class
Processes

## Rule TCL
```tcl
Rule {
	Initiator {
		Match PROCESS {
			Include OBJECT_NAME {
				-v "**"
			}
		}
    }
	Target {
		Match PROCESS {
			Include OBJECT_NAME { 					
				-v "aacinfo.exe"
			}
			Include -access "CREATE"
		}
	}
}
```
